## Cognition released a 34B coding model that beats Claude Sonnet on SWE-bench

Cognition (the Devin AI coding agent company) open-sourced FrontierCode, a 34-billion-parameter model trained specifically for agentic software engineering. It scores 28.6% on SWE-bench Verified, outperforming Claude 3.5 Sonnet and matching models 5x its size. The model is Apache 2.0 licensed and available on Hugging Face.

*A smaller, open-weight model now matches frontier performance on real-world coding tasks.*
